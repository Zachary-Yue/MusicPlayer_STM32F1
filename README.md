
# 《一起去未来吧》- 刘浩存

蜂鸣器演奏完整曲目哦！（总时长 3 分 12 秒左右）

## 引脚说明

| 蜂鸣器 | 按键 | LED 灯 |
|:---:|:---:|:---:|
| TIM2_CH1 | GPIO IPU | GPIO Out PP |
| `PA0` | `PA2` | `PC13` |

## 面包板电路连接

- 蜂鸣器：`PA0` 可以直接连接在无源蜂鸣器的**信号脚**上，无源蜂鸣器的 Vcc 可以串联一个**电位器**以调节音量。
- 按键：`PA2` 直接通过按键连接到低电平即可。
- LED：板载 LED，无需连接。
